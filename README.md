## Features

| Feature | Description |
|---|---|
| **Greeting** | Real-time clock, full date, and contextual greeting (morning/afternoon/evening/night) |
| **Custom Name** | Personalize the greeting with your name — saved automatically |
| **Focus Timer** | Pomodoro-style countdown with Start, Stop, and Reset controls |
| **Configurable Duration** | Set timer from 1 to 120 minutes — persisted across sessions |
| **To-Do List** | Add, edit inline, complete, delete, sort, and duplicate prevention |
| **Quick Links** | Save your favorite sites as one-click buttons |
| **Light / Dark Mode** | Toggle theme with OS preference fallback — no flash on load |
| **Local Storage** | All data saved in the browser — no account, no server |
